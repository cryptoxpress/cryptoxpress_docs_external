# Import Your Existing Wallet

<!-- IMPORT -->

import Link from "@docusaurus/Link";

<!-- BODY -->

You can also import your existing Wallet from other apps to **CryptoXpress.**

## What you'll need

You need to have your private key of the wallet to import it.

Check out the Video Walkthrough on how to view Private Key on metamask.

Now, To import the Wallet Just follow these steps

- **Step 1: Register for an account**
- **Step 2: Open Wallet Screen**
- **Step 3: Buy/Deposit crypto**
- **Step 4: Start trading**

### Step 1: Register for an account

Register for a CryptoXpress account from the App or website with your email.

### Step 2: Open Wallet Screen

On the top right corner you'll see wallet icon , tapping on it will take you to wallets section.

<center>
  <img src={"/img/basic_wallet_banner_1.png"} width="270" />
</center>

### Step 3: Open the Defi Tab

Under Crypto tab go to the DeFi tab tap on the Import Button

<center>
  <img src={"/img/basic_wallet_banner_2.png"} width="270" />
</center>

### Step 4: Import the Wallet

Now enter the Private Key and the wallet name that you want and then tap on Import.

  <img src={"/img/basic_wallet_banner_3.png"} width="270" />
</center>
